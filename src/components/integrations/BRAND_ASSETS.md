# Connector brand assets

- Figma, GitHub, Notion, Obsidian, and Framer use `simple-icons` v16.25.0 paths. Their monochrome SVG geometry is rendered with the current theme foreground for light/dark contrast. Simple Icons metadata and SVG paths are distributed under CC0-1.0; product marks remain trademarks of their owners.
- Canva uses the `bxl:canva` SVG distributed by [Iconify](https://iconify.design/) from the Boxicons Brands icon set under the MIT license. Upstream asset: `https://api.iconify.design/bxl:canva.svg`. It is locally bundled as `canva.svg`, rendered with the current theme foreground, and retains Canva's trademark status. SHA-256: `5e4df80c7318fdb5c464b51210b2a8eeacbaf1981010647c438f1cfa91c5ff1e`.
- Pencil uses the 180px Apple touch icon published by the official [pen.dev](https://pen.dev/) site. Upstream asset: `https://pen.dev/apple-touch-icon.png`. It is locally bundled as `pencil.png` under Pencil's trademark and brand terms. SHA-256: `738c8f4a0df8e7b561b0c620b48be68d429728a03b57a775e2afeb046dbff9dd`.
- Paper uses the 512px app icon published by the official [paper.design](https://paper.design/) site and declared there as the organization/application logo. Upstream asset: `https://paper.design/logos/app-icons/Paper%20App%20Icon%20512.png`. It is locally bundled as `paper.png` under Paper's trademark and brand terms. SHA-256: `957e2bd4fa8075a3ce94f94ad55a9028ff33f6e42c5cbe0bbb3eb87f2364ccbd`.
- Repository is the only labeled controlled fallback and is not represented as an official product logo.
- Assets are bundled locally. No runtime URL is loaded.
