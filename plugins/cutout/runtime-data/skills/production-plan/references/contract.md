# Production Plan contract

- Capability status: `internal-only`.
- Agent operation: `validate`.
- MCP tool: `cutout_validate`.
- Input: accepted outcome, current Design IR revision, explicit materials, policy and delivery intent.
- Output: a reviewable dependency DAG with deliverables, evidence requirements, effects and blockers.
- Planning performs no provider call, export, coding task or external write.
- Provider-required nodes remain blocked without an injected executor and applicable approval.
- Remote integrations require documented authorized hosts; OAuth, cloud collaboration and live sync remain roadmap until exposed by the capability manifest.
- Every completion claim requires authoritative artifacts or receipts from later execution.
